# 局タスク管理ボード

組合の局（局長 + 部下2名）で使う、シンプルなかんばん形式のタスク管理ツールです。
サーバー不要、`index.html` をブラウザで開くだけで動きます。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く（Chrome / Edge / Safari 推奨）
2. 初回起動時にこの端末用の合言葉（パスワード）を設定します
3. 列（未着手 / 進行中 / 完了）にカードをドラッグ&ドロップして進捗を管理します
4. カードをクリックすると、担当者・フェーズ・開始日・期限・チェックリスト・関連タスク・コメントを編集できます

## フェーズについて

「フェーズ」は、かんばんの列（ステータス）とは別に、案件・タスクを大きな期間や区分で分けておくための箱です。

- 画面上部の「フェーズ:」バーで、いま表示するフェーズを切り替えられます（📌は既定のフェーズ）
- 次の期間の作業や、着手を後回しにしたいタスクを別フェーズに置いておけば、今のフェーズの表示にノイズが出ません
- 完了・保留にしたフェーズは「アーカイブする」ことで絞り込みタブから隠せます（データは消えず、「アーカイブ済みも表示」で再確認可能）
- フェーズの作成・切替は画面上部のバー、名前変更・アーカイブ・削除は「⚙ 設定」から行います

## メンバー間でデータを共有する方法

このツールはサーバーを使わず、データはブラウザ内（localStorage）に保存されるため、
複数人・複数端末で自動的に同じデータが見えるわけではありません。以下のどちらかの方法で共有してください。

- **同じ端末・同じブラウザを共有する**（例: 局の共有PC）
- **書き出し(JSON) → 読み込み** で更新内容をやり取りする
  - 画面右上の「↓ 書き出し」でファイル（`kanban_YYYY-MM-DD.json`）をダウンロード
  - メール・LINE・共有フォルダなどで他のメンバーに渡す
  - 相手は「↑ 読み込み」でそのファイルを取り込む（現在のボードは上書きされます）

より頻繁に同期したい場合は、書き出したJSONをGoogleドライブなど共有フォルダに置き、
更新のたびに読み込みし直す運用がおすすめです。

## 主な機能

- 列とカードのドラッグ&ドロップによるかんばん管理（列は自由に追加・名称変更・削除可能）
- フェーズによるタスクの区分・絞り込み・アーカイブ
- カードごとの担当者・開始日・期限日の設定
- チェックリスト（サブタスク）とコメント
- 関連タスクの相互リンク（カード同士を紐づけ、クリックで行き来できます）
- カードごとの変更履歴（タイトル・担当者・フェーズ・日付・ステータス変更などを自動記録）
- 期限が近い/過ぎたカードを自動でハイライトし、画面上部にアラート表示
- ブラウザ通知（許可すれば期限超過・期限間近を通知）
- 設定画面からメンバー・フェーズ一覧の編集、期限アラートの基準日数変更、合言葉の変更が可能

## 注意事項（セキュリティについて）

- 合言葉によるロックは、この端末・このブラウザにおける簡易的な入室制限であり、
  強固なセキュリティ機構ではありません。機密性の高い情報の管理には使わないでください。
- データはブラウザのlocalStorageに保存されるため、ブラウザのデータを消去すると失われます。
  定期的に「書き出し」でバックアップを取ることをおすすめします。
